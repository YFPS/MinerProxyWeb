**MinerProxyWeb 基于ubuntu Redir 制作，用于简化Redir操作流程和管理，带web管理页面。带有定制可用于订阅收费的开源项目
---

## 功能

- 1.收益查询
+ 2.矿池代理
+ 3.自助兑换发卡
+ 4.过期自动处理

### 安装核心依赖 （Redir）

+ 1.下方命令复制粘贴至命令行回车执行

+ `sudo apt-get install redir -y`

### 启动

+ `
java -jar xxxx.jar
`
